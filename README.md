# Clover2D

Small 2D engine :)

* Rendering
  * OpenGL
  * Line rendering
  * Quad batch renderer
* ImGui
* Text rendering
* Sprite Animations
* Sprite Atlas
* Basic UI
* Delegates for custom events
* Logging
* Custom ECS (Entity Component System)
